🌡️ 第5条 AIの本気度について

本サービスでは、アプリ生成時にAIの「本気度（使用モデル）」がランダムに変わります。

正直に言うと：全員毎回Sonnetで生成するとAPIコストが続きません。だからランダムにしています。コスト制約を楽しい体験に変えた設計です。

🥋 通常（約84%）
Claude Haiku モデルを使用。手早く実用的なアプリを生成します。
生成URLは30日間有効です。

⚡ 師範直伝（約13%）
Claude Haiku モデル（強化プロンプト）を使用。UIや細部の品質を一段階引き上げます。
ローディング画面が水色に変化します。URLは60日間有効です。
機能追加ボタンが表示され、後から機能を足して育てられます。

🌟 Sonnet本気（約3%）
Claude Sonnet モデルを使用。デザイン・機能ともに丁寧な仕上がりで生成します。
ローディング画面が金色に変化します。URLは90日間有効です。
機能追加ボタンが表示され、後から機能を足して育てられます。

すべてのモードはユーザーに追加課金を求めることなく、ランダムに自動選択されます。

✨ 機能追加について（師範直伝・Sonnet本気限定）
師範直伝またはSonnet本気モードで生成されたアプリには、生成後に機能を追加できます。
・コード充填率80%未満の間、何回でも追加可能
・1日3回のレート制限にカウントしません
・追加生成のコストは運営者が負担します（無料）
📝 ご理解いただきたい点

・各モードの確率はシステム調整等により予告なく変動する場合があります
・上位モードが選ばれた場合でも、生成結果の品質を絶対的に保証するものではありません
・AIの性質上、生成結果には常にばらつきがあります
・確率を狙った連続実行は、1日3回の利用制限の範囲内でのみ可能です
・各モードの仕様・確率は予告なく変更・廃止される場合があります

🌡️ Article 5 - AI Effort Level

Each time you generate an app, the AI's "effort level" (model used) changes randomly.

Honest reason: running Sonnet for every user every time would make the service unsustainable. So we randomize it. It's a way of turning a cost constraint into an enjoyable experience.

🥋 Standard (~84%)
Uses Claude Haiku model. Quick and practical app generation.
Generated app URL is valid for 30 days.

⚡ Sensei Special (~13%)
Uses Claude Haiku model with enhanced prompting. Elevates UI quality and attention to detail.
Loading screen turns blue. URL valid for 60 days.
Feature Addition button appears — you can grow the app by adding features.

🌟 Sonnet Full Power (~3%)
Uses Claude Sonnet model. Top quality in both design and features.
Loading screen turns gold. URL valid for 90 days.
Feature Addition button appears — you can grow the app by adding features.

All modes are randomly selected with no additional charge to the user.

Apps generated in Sensei Special or Sonnet Full Power mode can have features added after generation.
· Available as long as code fill rate is below 80%
· Does not count toward the daily 3-generation limit
· Free of charge (costs covered by the operator)
